# Images Directory

This directory contains all the screenshots and images used in the README.

## Required Images:
- `portfolio-view.png` - Screenshot of the main portfolio dashboard
- `exchange-view.png` - Screenshot of the exchange interface  
- `exchange-detail.png` - Screenshot of the detailed exchange screen
- `demo-thumbnail.png` - Thumbnail image for the demo video
- `app-icon.png` - App icon (if needed)
- `chart-interaction.gif` - GIF showing chart interactions (optional)

## Image Guidelines:
- **Format**: PNG for screenshots, JPG for photos, GIF for animations
- **Resolution**: 1170x2532 (iPhone 14 Pro) or 1179x2556 (iPhone 15 Pro)
- **Quality**: High resolution, clear and crisp
- **Background**: Use actual app interface, no mockups needed

## How to Capture Screenshots:
1. Run the app on iPhone simulator or device
2. Navigate to the desired screen
3. Press `Cmd + S` (simulator) or `Volume Up + Side Button` (device)
4. Save images to this directory with appropriate names

## Video Thumbnail:
- Create an attractive thumbnail that represents the app
- Size: 1280x720 (16:9 aspect ratio)
- Include app name and key visual elements

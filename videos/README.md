# Videos Directory

This directory contains demo videos and screen recordings of the app.

## Required Videos:
- `app-demo.mp4` - Main demo video showing key features
- `chart-interactions.mp4` - Video demonstrating chart functionality (optional)
- `exchange-flow.mp4` - Video showing the complete exchange process (optional)

## Video Guidelines:
- **Format**: MP4 (H.264 codec recommended)
- **Resolution**: 1080p (1920x1080) or higher
- **Duration**: 2-5 minutes for main demo, 30-60 seconds for feature demos
- **Frame Rate**: 30fps or 60fps
- **Quality**: High quality, smooth playback
- **Audio**: Optional - can include background music or narration

## Recording Instructions:
1. **iOS Simulator**: Use QuickTime Player
   - Open QuickTime Player
   - File → New Screen Recording
   - Click the dropdown arrow next to record button
   - Select your iOS Simulator
   - Start recording and demonstrate the app

2. **Physical Device**: Use built-in screen recording
   - Control Center → Screen Recording
   - Or Settings → Control Center → Add Screen Recording

## Demo Video Structure:
1. **App Launch** (5-10 seconds)
2. **Portfolio Overview** (20-30 seconds)
3. **Chart Interactions** (20-30 seconds)
4. **Exchange Feature** (30-45 seconds)
5. **Navigation Flow** (20-30 seconds)
6. **Key Features Highlight** (30-45 seconds)

## Editing Tips:
- Keep transitions smooth
- Highlight key interactions with cursor/touch indicators
- Add text overlays for important features
- Trim unnecessary waiting time
- Export in web-optimized format for GitHub
